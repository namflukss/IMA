# מתכוני אמא

אפליקציית מתכונות אישית, בעברית (RTL), בנויה כ-PWA שאפשר להוסיף למסך הבית באנדרואיד.

כוללת: מסך בית לפי קטגוריות, רשימת מתכונים לקטגוריה, מסך מתכון (מרכיבים, שלבי הכנה, טיפ של אמא), מצב הכנה צעד-אחר-צעד, חיפוש, מועדפים, תכנון ארוחות/אירוחים, והוספת מתכון אוטומטית מצילום או מהדבקת טקסט (ראו למטה).

המידע (מתכונים, מועדפים, ארוחות מתוכננות) נשמר מקומית בדפדפן (localStorage) — אין צורך בשרת או חשבון, מלבד לתכונת המילוי האוטומטי.

## פיתוח מקומי

```bash
npm install
npm run dev
```

תכונת "מילוי אוטומטי" (צילום/הדבקת טקסט) לא תעבוד ב-`npm run dev` הרגיל — היא צריכה את הפונקציה השרתית ב-`api/`, שרצה רק דרך Vercel (ראו הפעלה מקומית מלאה למטה).

## בנייה לפרודקשן

```bash
npm run build
```

הפלט נכתב לתיקיית `dist/` — קבצים סטטיים בלבד.

## פריסה (Deploy) — Vercel

1. חברי את הריפו הזה ל-[Vercel](https://vercel.com/new) — הוא יזהה אוטומטית פרויקט Vite (build: `npm run build`, output: `dist`) וגם את התיקייה `api/` כפונקציה שרתית.
2. ב-Project Settings → Environment Variables, הוסיפי משתנה בשם `ANTHROPIC_API_KEY` עם מפתח API של Anthropic (מ-[console.anthropic.com](https://console.anthropic.com)). בלי זה, כפתורי "צילום מתכון" ו"הדבקת טקסט" יחזירו שגיאה.
3. Deploy — תקבלי לינק חי.

לאחר הפריסה, אמא יכולה לפתוח את הלינק בכרום באנדרואיד וללחוץ "הוספה למסך הבית" כדי שהאפליקציה תיפתח כמו אפליקציה רגילה.

### הפעלה מקומית מלאה (כולל המילוי האוטומטי)

כדי לבדוק את תכונת הצילום/הדבקה מקומית, צריך את ה-CLI של Vercel:

```bash
npm install -g vercel
vercel dev
```

זה יריץ גם את ה-frontend וגם את הפונקציה ב-`api/parse-recipe.js` יחד. יש להגדיר `ANTHROPIC_API_KEY` בקובץ `.env.local` בשורש הפרויקט.

## איך עובד המילוי האוטומטי

- **צילום מתכון**: מצלמים או בוחרים תמונה של כרטיס מתכון (כתוב יד או מודפס), התמונה נדחסת בדפדפן ונשלחת ל-`api/parse-recipe`, שמפעיל את Claude (Opus) כדי לקרוא את התמונה ולהחזיר מתכון מובנה (שם, קטגוריה, מנות, זמנים, רכיבים, שלבים, טיפ).
- **הדבקת טקסט**: מדביקים מתכון בכל פורמט חופשי, ואותה פונקציה שרתית מנתחת ומסדרת אותו.
- בשני המקרים השדות בטופס מתמלאים אוטומטית וניתן לערוך אותם לפני שמירה.
- עלות: כל קריאה למודל עולה כמה סנטים בודדים (חיוב לפי שימוש ב-Anthropic Console).

## מבנה הפרויקט

- `src/data/` — קטגוריות ומתכוני ברירת מחדל
- `src/state/useRecipeApp.js` — כל הלוגיקה (ניווט, CRUD, מצב הכנה, שמירה ל-localStorage)
- `src/screens/` — מסך לכל תצוגה באפליקציה
- `src/components/` — רכיבים משותפים (סרגל תחתון, אייקונים)
- `api/parse-recipe.js` — פונקציה שרתית (Vercel) שקוראת ל-Claude API לפענוח מתכונים
- `design/` — חומרי העיצוב המקוריים (Claude Design) ששימשו כבסיס למימוש
