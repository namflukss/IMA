# מתכוני אמא

אפליקציית מתכונות אישית, בעברית (RTL), בנויה כ-PWA שאפשר להוסיף למסך הבית באנדרואיד.

כוללת: מסך בית לפי קטגוריות, רשימת מתכונים לקטגוריה, מסך מתכון (מרכיבים, שלבי הכנה, טיפ של אמא), מצב הכנה צעד-אחר-צעד, חיפוש, מועדפים, ותכנון ארוחות/אירוחים.

המידע (מתכונים, מועדפים, ארוחות מתוכננות) נשמר מקומית בדפדפן (localStorage) — אין צורך בשרת או חשבון.

## פיתוח מקומי

```bash
npm install
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
```

הפלט נכתב לתיקיית `dist/` — קבצים סטטיים בלבד, אפשר לארח בכל שירות אחסון סטטי.

## פריסה (Deploy)

הכי פשוט: חברי את הריפו הזה ל-[Vercel](https://vercel.com/new) או [Netlify](https://app.netlify.com/start) — שניהם מזהים אוטומטית פרויקט Vite (build command: `npm run build`, output directory: `dist`), ותקבלי לינק חי תוך דקה.

לאחר הפריסה, אמא יכולה לפתוח את הלינק בכרום באנדרואיד וללחוץ "הוספה למסך הבית" כדי שהאפליקציה תיפתח כמו אפליקציה רגילה.

## מבנה הפרויקט

- `src/data/` — קטגוריות ומתכוני ברירת מחדל
- `src/state/useRecipeApp.js` — כל הלוגיקה (ניווט, CRUD, מצב הכנה, שמירה ל-localStorage)
- `src/screens/` — מסך לכל תצוגה באפליקציה
- `src/components/` — רכיבים משותפים (סרגל תחתון, אייקונים)
- `design/` — חומרי העיצוב המקוריים (Claude Design) ששימשו כבסיס למימוש
